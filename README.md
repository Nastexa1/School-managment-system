Project Description:
This project is a full functional School Management System developed using PHP and MySQL.
The system is a web-based application that helps manage school activities such as student
management, class management, attendance, grades, and user authentication.
All website contents are dynamically loaded from the database. The system implements
CRUD operations, session and cookie-based authentication, automatic session expiration,
and full form validation following proper coding and programming practices.

System Modules and Features:
- User Registration
- User Login and Logout
- Secure Dashboard (Control Panel)
- Add Student
- View Student
- Edit Student
- Delete Student
- Add Class
- Update Class
- Delete Class
- Attendance Management
- Grades Management
- User Profile Management
- Session and Cookie based authentication
- Automatic session expiration (after 5 minutes)
- Full form validation

Technologies Used:
- HTML
- Tailwind CSS
- PHP
- MySQL

Database Information:
Database Name: school_management

Database tables include:
- users
- students
- classes
- attendance
- grades
- profiles (and other related tables)

The database can be imported using the provided .sql file.

Authentication and Security:
- Sessions are used to control access to dashboard pages
- Cookies are used for "Remember Me" functionality
- Proper logout implementation with session and cookie destruction
- Automatic session expiration to enhance security
